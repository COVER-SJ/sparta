# 프로젝트 [AHA, anymore homeless animals 애니몰 홈리스 애니몰]

### '동물관리 사이트'에 등록된 주인을 잃어버려 보호소에서 보호 중인 동물과 동물을 잃어버린 주인들이 찾는 동물들을 한눈에 볼 수 있는 사이트를 구현한다.

#### 회원제를 통해서 신뢰성있는 사이트로 구성. 
#### 보호 중인 동물과 잃어버린 동물을 빠르게 찾아볼 수 있다.
